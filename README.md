![](faceless.png)

Faceless is a deepfake detection system using a convolutional neural network.

## Requirements
* Python 3

## Installation

### Cloning the repository

```sh
git clone https://github.com/ManhNho/Faceless.git
cd Faceless
```
### Installing dependencies

```sh
pip3 install -r requirements.txt
```

## Starting Faceless

```sh
python3 app.py
```
